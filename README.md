
# ML-Challenge

Resolución de challenge técnico para ML





## Run Locally

clonar el proyecto

```bash
  git clone https://github.com/javacha-ux/ML-challenge.git
```

Cambiar al directorio

```bash
  cd ML-challenge
```

Ejecutar script (desde Git Bash por ejemplo)

```bash
  sh  req-local.sh   '{ "dna": [ "CGC", "GTA", "GTC" ] }'

```
